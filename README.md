## 分类详情

### AI - 人工智能相关

- **New Skill (Manual Check Needed)** - Please check https://www.clawhub.ai/skills for updates

### Browser - 浏览器相关

- **web-search** - 使用 Brave Search API 进行网络搜索
- **web-fetch** - 从 URL 获取并提取可读内容（HTML 转 Markdown/文本）
- **browser-control** - 通过 OpenClaw 浏览器控制服务控制浏览器

### Calendar - 日程相关

- **cron-scheduler** - 管理 Gateway 定时任务和唤醒事件

### Developer - 开发相关

- **skill-creator** - 创建或更新 AgentSkills，包含脚本、引用和资源
- **shell-execution** - 执行 Shell 命令，支持 PTY、后台运行和超时控制
- **gateway-management** - 重启、配置或更新 Gateway 服务
- **canvas-control** - 控制节点画布（呈现/隐藏/截图）
- **node-control** - 发现和控制配对节点（状态/描述/相机/屏幕/位置/运行）

### File - 文件相关

- **file-operations** - 读取、写入、编辑文件内容
- **feishu-folder** - 列出飞书文件夹中的文档和子文件夹

### Image - 图片相关

- **image-analysis** - 使用配置的图像模型分析图像

### Memory - 记忆相关

- **memory-management** - 语义搜索和读取记忆文件（MEMORY.md, memory/*.md）

### Message - 消息相关

- **bluebubbles** - BlueBubbles 外部频道插件，支持 REST API 发送/探测和 Webhook 接收
- **session-management** - 列出、发送消息和获取其他会话/子代理的历史
- **feishu-doc** - 飞书文档操作（创建/读取/写入/更新/删除/追加）
- **feishu-app** - 列出当前飞书应用权限（Scopes）

### Other - 其他

- **示例 Skill (待更新)** - 这是一个示例，实际内容需要从网站获取

### Voice - 语音相关

- **voice-activation** - 语音唤醒词检测，支持自定义关键词
- **text-to-speech** - 将文本转换为语音并返回 MEDIA 路径
